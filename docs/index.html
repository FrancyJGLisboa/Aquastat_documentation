Migration of AQUASTAT Statistical Processes into the SWS

Francy Lisboa (ESSD - Methodological Innovation Team)

2019-03-16

Introduction

AQUASTAT - FAO Global’s Water Statistics System is migrating its statistical processes into the Corporate Statistical Working System (SWS). This online book documents the major steps that have been taken so far in regard to this migration. The book describes the main institutional actors involved in the migration as well as the AQUASTAT - SWS resources (code lists, datasets, data tables) that have been created to support the SWS plugins (R modules) to meet technical unit requirements. Furthermore, the SWS plugins are presented in the form of chapters providing detailed description of their workflows involving input -> processing -> output as well as results showing whether the modules have lived up to expectations.
